# ReplaceAllTypes

Replace the type of all properties in an interface.

## Example

<Source dark language="ts" code={`
import { ReplaceAllTypes } from "@baggie/typescript";

interface Car {
    model: string;
    diesel: boolean;
    wheels: number;
    doors: number;
}

type CarFunctions = ReplaceAllTypes<Car, () => string>;

// The type above matches the interface below

interface CarSpecificProperties {
    model: () => string;
    diesel: () => string;
    wheels: () => string;
    doors: () => string;
}
`} />
