# screenColors

Blend two colors together with the screen mode.

The result will be lighter, since we're working with an additive color model (RGB).
This function is the opposite of `multiplyColors()`.

> See https://en.wikipedia.org/wiki/Blend_modes

## Parameters

| Parameter  | Type     | Required | Description           |
| :--------- | :------- | :------: | --------------------- |
| **colorA** | `string` |    ✔     | The base color.       |
| **colorB** | `string` |    ✔     | The other base color. |
