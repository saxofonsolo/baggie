# clamp

Make sure a number is within the range of a minimum and maximum value.

If the number is smaller or larger, the nearest accepted value will be
returned.

## Parameters

| Parameter  | Type                           | Required |
| :--------- | :----------------------------- | :------: |
| **input**  | `number`                       |    ✔     |
| **minMax** | `{ min: number; max: number }` |    ✔     |
